Web Safe Color Testing



多种色彩的搭配使得 Web 页面漂亮非常，于是很难想象早期 Web 那只有黑白两色的单调面孔。

网页的色彩是 RGB 格式的，即用三个分别代表红色(Red)、绿色(Green)、蓝色(Blue)的色彩深度的数值来表示， 数值的范围在 0 - 255(十进制) 或 00 - FF(十六进制) 之间。

但并不是每种可能的数值组合所代表的颜色都非常适合在网页上使用，因为这里边有着所谓的 Web 安全色的问题。 这些安全色可以保证在不同的系统上都正常显示，而其他色彩则不能保证。因此如果色彩运用不当，就会影响网页整体色彩搭配的协调。
